## 🌟 Overview

BMW AI Experience is an intelligent recommendation web application that suggests the most suitable BMW model based on user preferences such as:

✔ Budget  
✔ Family Size  
✔ Usage Purpose  
✔ Luxury Preference  

The system uses a trained **Random Forest Machine Learning Model** to provide accurate premium recommendations.
